# typetheory

This repository contains an evolving platform for studying type theory using Jupyter notebooks and automated quizzes.

Learning material lives under `notebooks/`. Each document is stored as both a
Jupyter notebook (`.ipynb`) and a matching `py`-percent script (`.py`). Rendered
Markdown and HTML are generated from the notebook for easy viewing.

See [ROADMAP.md](ROADMAP.md) for the current development plan.
